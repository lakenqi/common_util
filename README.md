# easyUtil_timeGap
  发布时间间隔显示
<h3>简介</h3>
		<h4>&emsp;&emsp;js实现发布时间间隔显示，可根据当前时间显示“刚刚”，“X小时前”等，引入js即可</h4>
		<h4>&emsp;&emsp;支持模块化引用</h4>
		<h3>API:</h3>
		<h3>入口</h3>
		<ul>
			<li>普通调用全局引用名称为easy_TimeGap，模块化可自定义名称</li>
		</ul>
		<h3>具体方法及参数（以普通调用名称为例）</h3>
		<ul>
			<li>
				以如下方法初始化即可：<br/>
				&emsp;&emsp;easy_TimeGap.create(时间戳（number)),返回值即为要显示的内容<br/>
			</li>
		</ul>
		<h4>兼容性：自测IE10+,FF,Chrome及其他浏览器兼容（IE10以下未测试）</h4>
